# 行为准则

[English](CODE_OF_CONDUCT.md)

Movscript 遵循 Contributor Covenant Code of Conduct 2.1。

## 我们的承诺

我们承诺让所有人都能在没有骚扰的环境中参与本项目，不因年龄、体型、残障、族裔、性别特征、性别认同与表达、经验水平、教育、社会经济状态、国籍、外貌、种族、宗教、性取向或身份而受到区别对待。

## 期望行为

- 保持尊重和建设性。
- 把批评聚焦在想法、代码和结果上。
- 假设善意，同时为自己造成的影响负责。
- 帮助维护对新贡献者友好的空间。

## 不可接受行为

- 骚扰、侮辱或歧视性语言。
- 公开或私下威胁。
- 未经许可发布他人隐私信息。
- 持续扰乱讨论或维护者工作。

## 执行

当行为损害项目社区时，维护者可以删除评论、关闭 issue、拒绝贡献或限制参与。
