# 架构

Movscript 是 local-first 桌面工作台，主要由四个应用组成：

- 桌面前端：Electron + React，负责用户工作流、项目上下文和本地服务启动。
- Go 后端：项目、资源、用户、任务、模型配置和生成任务的事实来源。
- 管理后台：配置凭据、模型、功能路由、用户、组织、资源和调试信息。
- 本地 Agent：独立 Node 服务，负责线程、运行、计划、工具策略和本地草案。

运行时边界：

- 后端保存正式项目数据和供应商凭据。
- 桌面端持有当前 UI 上下文，并通过本地 MCP-shaped endpoint 暴露给 Agent。
- Agent 保存本地运行状态、记忆、草案和 trace。
- 管理后台通过后端 API 操作系统级配置。
